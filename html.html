Listado De Etiquetas HTML
Comando Funcion
[html:5] Este nos genera la estructura basica de html.
[head - /head] Esta solo es visible para navegadores, nos permite saber datos como los enlaces, links, etc.
[link - /link] Se utiliza para afiliar datos externos al documento.
[body - /body] Este incluye todo el contenido que deseemos mostrar en la pagina.
[h1]-[h2]-[h3]-[h4]-[h5]-[h6] Nos permien colocar titulos a la pagina, con tamaño de letra de manera descendiente.
[p - /p] Nos permite introducir parrafos dentro de la pagina.
[a - /a] Nos permite crear hipervinculos dentro de la pagina.
[strong - /strong] Esta nos permite definir la importancia de uno o mas textos.
[br - /br] Nos permite realizar un salto de linea transparente dentro de la pagina.
[hr - /hr] Nos permite realizar un salto de linea, permitiendo visualizarlo con un vector horizontal.
[table - /table] Nos permite crear una tabla dentro de la pagina web.
[tr - /tr] Nos permite definir las celdas que posee una tabla.
[td - /td] Nos permite definir una celda en especifico de la tabla.
[img] Nos permite añadir imagenes en la web.
[ol - /ol] Nos permite crear una lista ordenada
[n - /n] Nos permite crear un texto en negrilla.
[div - /div] Este nos permite agrupar elementos en secciones.
[center - /center] Nos permite centrar el obejto o texto que deseemos.
[button - /button] Nos permite crear un boton dentro de la web.
[border] Nos permite darle una cantidad de borde a nuestros elementos.
[width - height] Nos permite dar un alto y un ancho a la imagen que deseemos modificar.
[class=""] Nos permite nombrar a un elemento por separado.
